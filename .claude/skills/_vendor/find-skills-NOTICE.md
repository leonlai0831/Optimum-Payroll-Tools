# Vendored skill — vercel-labs/skills · find-skills

`.claude/skills/find-skills/` is vendored verbatim from
[vercel-labs/skills](https://github.com/vercel-labs/skills) (MIT — declared in the
upstream `package.json` and README; the repo ships no standalone LICENSE file),
pinned to commit `be0dd25b4a8665894a56f45ef582cc02ca802c39`.

**What it is:** the discovery skill for the open agent-skills ecosystem — it
guides searching for and installing skills via the `npx skills` CLI
(`skills find` / `skills add`) and the skills.sh leaderboard.

**Why vendored (not bootstrapped):** MIT is permissive, so — like the
obra/superpowers, ui-ux-pro-max, and skill-creator vendors — committing it makes
it available in every Claude Code on the web session with zero network.

To update: re-copy from a newer vercel-labs/skills checkout and bump the commit above.
